# CYD (Cheap Yellow Display) 2.8 Inch

One of the most common screens for hobbyists is the CYD (Cheap Yellow Display). The original board has spawned a number of derivatives that all generally follow the same form factor, but have different internal code and wiring. 

We primarily use the FREENOVE ESP32-S3 2.8 Inch IPS 240x320 screen when referring to a "CYD", but code is also developed for other CYD variants like the DIYMalls ESP32-2432S028R.
